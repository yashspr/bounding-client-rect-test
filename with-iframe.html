With Iframe, lets observe the behaviour of bounding client rect

To perform this test, load two servers with the current folder as root. One on port 3000 & another on port 3001

And open this webpage from the link http://localhost:3000/with-iframe.html

This is when the iframe has a fixed width and height

So, the height and width that is set to the iframe acts as the viewport for the webpage embedded inside the iframe

This is when the iframe is using iframe resizer to get maximize to full width and height of the webpage in the iframe

since the height of iframe will be equal to the max height of the webpage in the iframe, it means the viewport = the webpage dimensions and there are no scroll bars. So the values in boundingClientRect doesnt change when scrolling
